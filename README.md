# Air Quality Index Prediction & Analysis

##  Project Overview
This project focuses on analyzing air pollution data and predicting the Air Quality Index (AQI) using machine learning techniques. The goal is to understand the relationship between different pollutants and AQI and to build a predictive model based on historical data.

##  Objective
- Perform data analysis on air quality data  
- Identify key pollutants affecting AQI  
- Build a machine learning model to predict AQI  
- Evaluate model performance  
- Visualize pollution trends and insights  

##  Dataset Description
The dataset contains air pollution parameters such as:

- CO AQI Value  
- Ozone AQI Value  
- NO2 AQI Value  
- PM2.5 AQI Value  
- Overall AQI Value
